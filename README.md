# BeautyLink
Plataforma Digital de Apoio a Empreendedoras da Estética em Comunidades Periféricas

## Estrutura do Projeto
O projeto está organizado nas seguintes pastas:

projeto-estetica/
frontend/
- interface da aplicação (HTML, CSS, JavaScript)
backend/
- lógica do servidor e integração com banco de dados
database/
- estrutura e scripts do banco de dados
docs/
- documentos do projeto (plano de ação, relatórios)
README.md
- documentação principal do projeto

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Framework Web
- Banco de Dados
- Git e GitHub para controle de versão

## Objetivo do Projeto
Desenvolver uma plataforma digital que permita a profissionais autônomas da área da estética divulgar seus serviços e organizar agendamentos de forma simples e acessível.
